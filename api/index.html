⚠️ Important: This simulation shows only frequency sensitivity loss — the most basic aspect of hearing loss. Real hearing loss also includes distortion, tinnitus (internal noise), fatigue, and difficulty understanding speech in noise — none of which can be simulated with simple filters. This is an approximation based on real data, not a clinical tool.

About the data

Source: NHANES (National Health and Nutrition Examination Survey), CDC/NCHS. 26,583 audiograms from 9 cycles (1999–2020).

Method: HDBSCAN density-based clustering with row-centering (preserves curve shape, removes overall level). 14 pure-tone thresholds (500–8000 Hz, bilateral).

Validation: 100× bootstrap (ARI median 0.68). External validation on OHHR dataset (581 people, Germany).

Source code: github.com/gabrielviniciusnascimento/the_frequency_ml (MIT License)

Disclaimer: This tool is for educational and empathy purposes only. It does not diagnose hearing loss. If you have concerns about your hearing, consult an audiologist.
